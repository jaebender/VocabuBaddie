# Third-Party Data Attribution

This project uses third-party vocabulary and dictionary data from the following sources.

## Wordset Dictionary

Repository:
https://github.com/wordset/wordset-dictionary

The dictionary definitions included in this project are derived from the Wordset Dictionary project.

Wordset by Wordset, Inc. is licensed under the **Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0)** license.

This project uses a filtered and preprocessed subset of the original dataset for educational purposes. The dataset was modified by selecting entries and preparing them for gameplay.

The Wordset dataset incorporates WordNet 3.0 data. The required WordNet copyright notice and license are included with this project.

## High Frequency Vocabulary

Repository:
https://github.com/arstgit/high-frequency-vocabulary

Word frequency rankings used during preprocessing were derived from the High Frequency Vocabulary project.

The High Frequency Vocabulary project is licensed under the **MIT License**. The frequency rankings were used to organize vocabulary into relative difficulty levels for gameplay.

## Modifications

The datasets from these projects were combined and preprocessed to create a vocabulary dataset suitable for a multiple-choice educational word game. Processing included filtering entries, selecting usable definitions, and preparing data for gameplay.

This project was created for educational purposes as part of a software engineering course.
