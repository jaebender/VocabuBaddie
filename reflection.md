# Reflection

As a practicing data engineer, many topics in this course - and especially this assignment are either new or untested by me. On a conceptual level, the practice of loop engineering was novel to me. Typically I don't utilize agentic workflows in my daily work, so understanding the modern software engeineer's setup was really useful. Learning to break down larger workflows in the various steps and stages demonstrated in class is a powerful framework I plan to adopt. On a technical level, this gave me practice using Codex in my IDE (and troubleshooting permissions as an added benefit.) Working alongside Codex in this assignment provided more repetitions to mimic a typically software engineer's workflow and understand the actual code implementation of a small project.
